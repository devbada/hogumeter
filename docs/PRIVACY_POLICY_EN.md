# Privacy Policy

**HoguMeter**
**Last Updated:** December 12, 2025
**Effective Date:** December 12, 2025

## 1. Overview

HoguMeter (the "App") is an **entertainment-purpose taxi meter simulation app** designed to make driving with friends more fun.

This Privacy Policy explains what information the App collects, how it is used, and how it is protected.

---

## 2. Information We Collect

### 2.1 Location Information

| Item | Details |
|------|---------|
| **Purpose** | Measure travel distance and calculate fares |
| **Data Collected** | GPS coordinates (latitude, longitude), travel speed |
| **When Collected** | Only while the meter is running |
| **Storage Location** | On user's device only (local storage) |
| **External Transmission** | **None** |

**Important:** Location data is processed **only on your device** and is never transmitted to external servers.

### 2.2 Trip History

| Item | Details |
|------|---------|
| **Purpose** | Allow users to view past trips |
| **Data Collected** | Start/end time, distance, calculated fare, region names |
| **Storage Location** | On user's device (UserDefaults) |
| **Retention Period** | Until deleted by user |
| **External Transmission** | **None** |

### 2.3 App Settings

| Item | Details |
|------|---------|
| **Purpose** | Maintain user preferences |
| **Data Collected** | Fare settings, dark mode, sound settings, disclaimer agreement |
| **Storage Location** | On user's device (UserDefaults) |
| **External Transmission** | **None** |

## 3. Information We Do NOT Collect

HoguMeter does **NOT** collect:

- Personal identification information (name, email, phone number)
- Account information (no sign-up required)
- Payment information
- Data from other apps (contacts, photos, calendar)
- Advertising identifiers (IDFA)
- Device unique identifiers
- Usage analytics data

## 4. How We Use Information

Collected information is used **only for**:

| Purpose | Description |
|---------|-------------|
| **Fare Calculation** | Calculate simulated taxi fares based on GPS distance and time |
| **Trip History** | Store trips for user reference |
| **Settings Persistence** | Maintain user's fare and app settings |

## 5. Data Storage and Security

### 5.1 Storage Location

All data is stored **only on the user's device**.

- No external servers
- No cloud synchronization
- No network communication (except for location permission)

### 5.2 Security

- Protected by iOS sandbox environment
- Inaccessible to other apps
- Protected by device encryption if device is locked

---

## 6. Data Deletion

### 6.1 Delete Trip History

- **Individual**: History > Select trip > Delete
- **All**: History > Delete All

### 6.2 Reset App Settings

- Settings > App Info > Reset Disclaimer

### 6.3 Delete All Data

- **Uninstalling the app** completely removes all data from the device

---

## 7. Location Permission

### 7.1 Permission Types

| Permission | Purpose |
|------------|---------|
| **While Using** | Measure distance when meter is active |
| **Always** | Continue measuring when screen is off |

### 7.2 Managing Permissions

Users can change location permissions anytime in iOS Settings:

```
Settings > Privacy & Security > Location Services > HoguMeter
```

**Options:**
- **Never**: Disable location (limits app functionality)
- **While Using**: Collect location only when app is open
- **Always**: Collect location in background (needed for continuous metering)
