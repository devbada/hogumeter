# 개인정보 처리방침

**호구미터 (HoguMeter)**
**최종 수정일:** 2025년 12월 12일
**시행일:** 2025년 12월 12일

---

## 1. 개요

호구미터(이하 "앱")는 친구들과 함께하는 드라이브를 더 재미있게 만들어주는 **엔터테인먼트 목적의 택시미터기 시뮬레이션 앱**입니다.

본 개인정보 처리방침은 앱이 어떤 정보를 수집하고, 어떻게 사용하며, 어떻게 보호하는지 설명합니다.

---

## 2. 수집하는 정보

### 2.1 위치 정보

| 항목 | 내용 |
|------|------|
| **수집 목적** | 이동 거리 측정 및 요금 계산 |
| **수집 항목** | GPS 좌표 (위도, 경도), 이동 속도 |
| **수집 시점** | 미터기 작동 중에만 수집 |
| **저장 위치** | 사용자 기기 내 (로컬 저장) |
| **외부 전송** | **전송하지 않음** |

**중요:** 위치 정보는 **오직 사용자의 기기 내에서만** 처리되며, 외부 서버로 전송되지 않습니다.

### 2.2 주행 기록

| 항목 | 내용 |
|------|------|
| **수집 목적** | 과거 주행 기록 조회 |
| **수집 항목** | 출발/도착 시간, 이동 거리, 계산된 요금, 출발/도착 지역명 |
| **저장 위치** | 사용자 기기 내 (UserDefaults) |
| **보관 기간** | 사용자가 삭제할 때까지 |
| **외부 전송** | **전송하지 않음** |

### 2.3 앱 설정

| 항목 | 내용 |
|------|------|
| **수집 목적** | 사용자 맞춤 설정 유지 |
| **수집 항목** | 요금 설정, 다크모드 설정, 효과음 설정, 면책 동의 여부 |
| **저장 위치** | 사용자 기기 내 (UserDefaults) |
| **외부 전송** | **전송하지 않음** |

---

## 3. 수집하지 않는 정보

호구미터는 다음 정보를 **수집하지 않습니다**:

- 이름, 이메일, 전화번호 등 개인 식별 정보
- 계정 정보 (회원가입 기능 없음)
- 결제 정보
- 연락처, 사진, 캘린더 등 다른 앱 데이터
- 광고 식별자 (IDFA)
- 기기 고유 식별자
- 사용 행태 분석 데이터

---

## 4. 정보의 사용 목적

수집된 정보는 **오직 다음 목적으로만** 사용됩니다:

| 목적 | 설명 |
|------|------|
| **요금 계산** | GPS 기반 이동 거리 및 시간 측정으로 가상의 택시 요금 계산 |
| **주행 기록** | 사용자가 과거 주행 내역을 확인할 수 있도록 저장 |
| **설정 유지** | 사용자가 설정한 요금 체계 및 앱 설정 유지 |

---

## 5. 정보의 저장 및 보안

### 5.1 저장 위치

모든 데이터는 **사용자의 기기 내에만** 저장됩니다.

- 외부 서버 없음
- 클라우드 동기화 없음
- 네트워크 통신 없음 (위치 권한 외)

### 5.2 보안

- iOS의 샌드박스 환경에서 안전하게 보호됩니다
- 다른 앱에서 접근할 수 없습니다
- 기기 분실 시 iOS의 기기 암호화로 보호됩니다

---

## 6. 정보의 삭제

### 6.1 주행 기록 삭제

- **개별 삭제**: 기록 > 기록 선택 > 삭제
- **전체 삭제**: 기록 > 전체 삭제

### 6.2 앱 설정 초기화

- 설정 > 앱 정보 > 면책 동의 초기화

### 6.3 모든 데이터 삭제

- **앱 삭제** 시 모든 데이터가 기기에서 완전히 삭제됩니다

---

## 7. 위치 정보 권한

### 7.1 권한 유형

| 권한 | 용도 |
|------|------|
| **앱 사용 중 위치** | 미터기 작동 시 거리 측정 |
| **백그라운드 위치** | 화면 꺼진 상태에서도 거리 측정 지속 |

### 7.2 권한 관리

사용자는 언제든지 iOS 설정에서 위치 권한을 변경할 수 있습니다:

```
설정 > 개인정보 보호 및 보안 > 위치 서비스 > 호구미터
```

**옵션:**
- **안 함**: 위치 기능 비활성화 (앱 기능 제한)
- **앱을 사용하는 동안**: 앱 실행 중에만 위치 수집
- **항상**: 백그라운드에서도 위치 수집 (미터기 연속 사용 시 필요)

---

## 8. 제3자 제공

호구미터는 수집된 정보를 **제3자에게 제공하지 않습니다**.

- 광고 네트워크 없음
- 분석 서비스 없음
- 외부 SDK 없음

---

## 9. 아동의 개인정보

호구미터는 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 앱은 개인 식별 정보를 수집하지 않으므로 연령 확인 절차가 없습니다.
