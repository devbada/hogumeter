Privacy Policy

Last Updated: December 12, 2025

1. Overview

HoguMeter (the "App") is an entertainment-purpose taxi meter simulation app designed to make driving with friends more fun.

This Privacy Policy explains what information the App collects, how it is used, and how it is protected.

2. Information We Collect

2.1 Location Information

ItemDetails
PurposeMeasure travel distance and calculate fares
Data CollectedGPS coordinates (latitude, longitude), travel speed
When CollectedOnly while the meter is running
Storage LocationOn user's device only (local storage)
External TransmissionNone
Important: Location data is processed only on your device and is never transmitted to external servers.

2.2 Trip History

ItemDetails
PurposeAllow users to view past trips
Data CollectedStart/end time, distance, calculated fare, region names
Storage LocationOn user's device (UserDefaults)
Retention PeriodUntil deleted by user
External TransmissionNone

2.3 App Settings

ItemDetails
PurposeMaintain user preferences
Data CollectedFare settings, dark mode, sound settings, disclaimer agreement
Storage LocationOn user's device (UserDefaults)
External TransmissionNone

3. Information We Do NOT Collect

HoguMeter does NOT collect:

  • Personal identification information (name, email, phone number)
  • Account information (no sign-up required)
  • Payment information
  • Data from other apps (contacts, photos, calendar)
  • Advertising identifiers (IDFA)
  • Device unique identifiers
  • Usage analytics data

4. How We Use Information

Collected information is used only for:

PurposeDescription
Fare CalculationCalculate simulated taxi fares based on GPS distance and time
Trip HistoryStore trips for user reference
Settings PersistenceMaintain user's fare and app settings

5. Data Storage and Security

5.1 Storage Location

All data is stored only on the user's device.

  • No external servers
  • No cloud synchronization
  • No network communication (except for location permission)

5.2 Security

  • Protected by iOS sandbox environment
  • Inaccessible to other apps
  • Protected by device encryption if device is locked

6. Data Deletion

6.1 Delete Trip History

  • Individual: History > Select trip > Delete
  • All: History > Delete All

6.2 Reset App Settings

Settings > App Info > Reset Disclaimer

6.3 Delete All Data

Uninstalling the app completely removes all data from the device.

7. Location Permission

7.1 Permission Types

PermissionPurpose
While UsingMeasure distance when meter is active
AlwaysContinue measuring when screen is off

7.2 Managing Permissions

Users can change location permissions anytime in iOS Settings:

Settings > Privacy & Security > Location Services > HoguMeter

8. Third-Party Sharing

HoguMeter does NOT share information with third parties.

  • No advertising networks
  • No analytics services
  • No external SDKs

9. Children's Privacy

HoguMeter does not knowingly collect personal information from children under 14. Since the app does not collect personally identifiable information, no age verification is required.

10. Changes to This Policy

This Privacy Policy may be updated due to legal changes or app feature updates.

  • Changes will be communicated through app update notes or in-app notices
  • Significant changes may require re-consent upon app launch
